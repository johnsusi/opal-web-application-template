Opal web-application template
================================================================================

Getting started
--------------------------------------------------------------------------------

```
bundle install
bundle exec rackup
```

Browse application from [http://localhost:9292/]. It will rebuild on changes.

Build the source manually
--------------------------------------------------------------------------------

```
rake build
```
